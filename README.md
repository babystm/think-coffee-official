# SYMMETRY Coffee Shop Website

A premium, minimalist static website created with HTML, CSS, and Vanilla JavaScript.

## Features
-   Responsive design
-   Modern typography (Playfair Display & Inter)
-   Scroll reveal animations
-   Premium aesthetic matching the SYMMETRY brand
-   Deployment-ready for GitHub Pages

## Deployment
This website is ready to be hosted on GitHub Pages. Simply push these files to a GitHub repository and enable GitHub Pages in the repository settings.
